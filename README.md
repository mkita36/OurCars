# README

* 概要  
　　社用車管理アプリ  
　　実際の会社の担当者にヒアリングのうえ開発  
　　（現在実際に利用いただいています）  
　・社用車の実績や車検情報等を包括的に管理  
　・実績・管理情報のエクセル出力  
　・給油・オイル交換等のアラート機能  
  
* 機能・技術一覧  
　・認証機能：　　　　　　 　 devise  
　・PWA：　　　　　　　　　　serviceworker-rails  
　・エクセル出力：　　　　　　caxlsx  
　・予約機能：　　　　　　　　simple_calender
    
* その他、使用している技術  
　・テスト：　　　　rspec  
　・データベース：　postgreSQL  
　・インフラ：　　　heroku  
　・html：　　　　  slim  
　・CSS：　　　　　Bootstrap,  
　　　　　　　　　　レスポンシブデザイン
